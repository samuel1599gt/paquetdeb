# PaketiQ 🔍📦

**PaketiQ** es una herramienta avanzada de línea de comandos (CLI) diseñada para el análisis profundo de flujos de red y la detección de anomalías. 

Este proyecto ha sido íntegramente desarrollado por **Cyber and Code** para proporcionar una solución técnica eficiente en entornos de ciberseguridad y auditoría de redes.

---

## 🚀 Características principales

* **Extracción de flujos:** Procesa archivos PCAP para identificar flujos de red con precisión.
* **Análisis de Topología:** Mapea conexiones y relaciones entre nodos de red.
* **Detección de Anomalías:** Implementa lógica de preprocesamiento para identificar comportamientos inusuales.
* **Generación de Reportes:** Exporta resultados detallados listos para análisis forense.
* **Distribución Nativa:** Disponible como paquete binario para una integración total con el sistema.

---
